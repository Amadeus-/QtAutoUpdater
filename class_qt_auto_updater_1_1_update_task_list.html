Detailed Description

A simple list of tasks to be handeled sequentially.

The UpdateTaskList represents a list of tasks. These tasks can be any kind of UpdateTasks. The list itself will simply handle one task after another. This means all added tasks (except of the last one) should not be infinite, because this way the list would stick with that task forever.

Note
This task will be stored automatically. It will store all the task in the list that are storeable. If a task is not, it will simply be discarded.

The tasklist will be empty. Since this class inherits QLinkedList, you can add tasks using for example QLinkedList::append.
